# Hospital Management System (HMS)

## Technologies Used
- Java (Swing GUI)
- JDBC
- MySQL Workbench
- NetBeans IDE

## Setup Instructions
1. Import SQL schema into MySQL Workbench.
2. Open project in NetBeans.
3. Edit DBConnection.java with your database username/password.
4. Build and run the application.

## Features
- Schedule, update and cancel Appointments
- View records in a GUI
- Update and delete via table selection
